通过查看native-lib.cpp方法，stringFromJNI目的是向Java层返回一个字符串。如果要在native-lib.cpp文件中添加新的方法，必须添加在extern”C” { } 中，或者在每个方法前加extern”C”, 否则会报找不到方法。如果源文件为C，则须将extern“C”部分去掉，因为extern “C”的作用就是告诉编译器以C方式编译。
